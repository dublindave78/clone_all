# CloneRepo
 Clone all user repositories from Github

 `using only username`

## Installation
```
npm install @wael14/github_clone_all
```

## Usage
```
const clone_all = require('./github_clone_all');

clone_all.username_clone_all('tecno14');
```

you can put this two line in `file.js` then run it:
```
node file.js
```
